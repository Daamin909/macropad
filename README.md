### Specifications

BOM:

- 11x Blank DSA Keycaps (6x Black, 3x White, 2x Red)
- 11x Cherry MX Switches
- 12x 1N4148 diodes
- 1x XIAO RP2040
- 4x M3x16 Bolt
- 4x M3 Heatset
- 1x SSD1306 128x32 0.91" OLED (5V VCC, 3.3V logic, I2C)
- 1x EC11 Rotary Encoder
- 1x PCB
- 1x Case (Top and Bottom)

Others:

- KMK Firmware
- TOP.step
- BOTTOM.step
